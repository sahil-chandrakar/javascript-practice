# JavaScript Practice Workspace

This repository is for practicing and experimenting with JavaScript, including tasks, scripts, and styles. It is structured for easy learning and testing of JavaScript concepts.

## Project Structure

- `index.html` – Main HTML file
- `assets/script/task.js` – JavaScript code for tasks
- `assets/style/style.css` – Stylesheet

## How to Use

1. Open `index.html` in your browser to see the project in action.
2. Edit the JavaScript and CSS files in the `assets` folder to experiment with code.
